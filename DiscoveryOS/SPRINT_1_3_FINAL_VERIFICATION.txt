================================================================================
ROOT CAUSE OF FAILURES
================================================================================

1. TypeScript 7.0.2 Incompatibility
   - Breaking change: New moduleResolution defaults
   - ts-node 10.9.2 couldn't initialize TypeScript system
   - Error: "Cannot read properties of undefined (reading 'fileExists')"
   - Fix: Downgrade to TypeScript 5.9.3

2. Incorrect moduleResolution Setting
   - Previous: "bundler" (unsupported by ts-node 10.x)
   - ts-node expects "node" for compatibility
   - Fix: Changed to "moduleResolution": "node"

3. @nestjs/config Version Incompatibility
   - Version 3.3.0 requires NestJS 8-10 (peer dependency)
   - NestJS 11 installation uses version 3.3.0 by default
   - Causes unresolved peer dependency conflicts
   - Fix: Explicitly set @nestjs/config to 3.2.0 (NestJS 11 compatible)

4. Missing "npm run dev" Script
   - Only "start:dev" was defined, not "dev"
   - Fix: Added "dev" script for convenience
